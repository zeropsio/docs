* `3.17`
* `3.18`
* `3.19`

* `22.04`
* `ubuntu@latest`

* `dotnet@6`
* `dotnet@7`
* `dotnet@latest`

* `go@1.22`
* `go@latest`
* `golang@1`
* `golang@latest`
### nginx
* `nginx@1.22`
* `nginx@latest`

* `nodejs@20`
* `nodejs@22`
* `nodejs@latest`

* `php-apache@8.1`

* `php-nginx@latest`

* `python@3.11`
* `python@3.12`
* `python@latest`
### Rust
* `rust@1.78* `
