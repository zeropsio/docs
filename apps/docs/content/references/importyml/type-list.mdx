This is a list of all supported service types that can be used in import yaml configuration file.

## Node.js

- `nodejs@18`
- `nodejs@20`
- `nodejs@22`

## Python

- `python@3.11`
- `python@3.12`

## PHP & Apache

- `php-apache@8.1`
- `php-apache@8.3`

## PHP & nginx

- `php-nginx@8.1`
- `php-nginx@8.3`

## Go

- `go@1`

## Rust

- `rust@1`

## .NET

- `dotnet@6`
- `dotnet@7`

## KeyDB

- `keydb@6`

## PostgreSQL

- `postgresql@12`
- `postgresql@14`
- `postgresql@16`

## MariaDB

- `mariadb@10.6`

## Static service

- `nginx@1.22`
- `static`

## Object storage

- `object-storage`

## Shared storage

- `shared-storage`
