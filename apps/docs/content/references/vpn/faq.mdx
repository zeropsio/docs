<FAQ>
  <FAQItem question="Why is my VPN in WSL2 not working">
    <p>
      The reason can be that <code>systemd</code> is not running in WSL2 by
      default. To fix this:
      <ol>
        <li>
          Run <code>sudo -e /etc/wsl.conf</code>
        </li>
        <li>
          Add <code>system=true</code> to <code>[boot]</code> section
        </li>
        <li>
          Comment out the first line{' '}
          <code>LABEL=cloudimg-rootfs / ext4 defaults 0 1</code>
        </li>
        <li>
          In <code>cmd.exe/PowerShell</code> run <code> wsl --shutdown</code> to
          restart WSL2.
        </li>
      </ol>
    </p>
  </FAQItem>
</FAQ>
