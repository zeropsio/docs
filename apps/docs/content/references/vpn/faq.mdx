  <FAQItem question="Why is my VPN in WSL2 not working">
    <p>
      The reason can be that `systemd` is not running in WSL2 by
      default. To fix this:
      <ol>
        <li>
          Run `sudo -e /etc/wsl.conf`
        </li>
        <li>
          Add `system=true` to `[boot]` section
        </li>
        <li>
          Comment out the first line{' '}
          `LABEL=cloudimg-rootfs / ext4 defaults 0 1`
        </li>
        <li>
          In `cmd.exe/PowerShell` run ` wsl --shutdown` to
          restart WSL2.
        </li>
      </ol>
    </p>
  </FAQItem>
