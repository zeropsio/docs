Development, deploy, verification, and evidence-driven fixes are part of the normal ZCP development lifecycle. The user-facing development controls live in [Build with ZCP → Develop with live project context](/zcp/workflows/build-with-zcp#develop-with-live-project-context). The deeper verification model lives in [How ZCP works → Verification has two layers](/zcp/concept/how-it-works#verification-has-two-layers).

Those sections explain how the agent works from live project state, Zerops-specific knowledge, project-scoped tools, deploy evidence, and behavior proof.

For symptoms and recovery moves, use [Troubleshooting](/zcp/reference/troubleshooting).
