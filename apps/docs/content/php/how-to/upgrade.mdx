---
title: How to upgrade the PHP version
---

You can upgrade or downgrade your PHP service to a different major PHP version by setting the <a href="/php/how-to/build-pipeline#base-1">`run.base`</a> parameter in your `zerops.yml`. When you [trigger a new pipeline](/php/how-to/trigger-pipeline), Zerops will start new runtime container(s) with the required PHP version. If you don't specify the `run.base` attribute in your `zerops.yml`, Zerops keeps the current PHP version for your runtime.

If you want to build your application with a different major PHP version, change the <a href="/php/how-to/build-pipeline#base">`build.base`</a> parameter in your `zerops.yml`. The `build.base` is the required attribute.
