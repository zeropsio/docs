# MariaDB overview

[MariaDB ↗](https://mariadb.org/) is one of the most popular open source relational databases. It’s made by the original developers of MySQL and guaranteed to stay open source.

## How to start

Need a hint? Feel like skipping the theory and diving in practical examples first? Check out these recipes containing a MariaDB service. Feel free to fork any repository and try what you can do with Zerops.
